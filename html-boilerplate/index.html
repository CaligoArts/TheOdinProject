Main thing's to know about HTML is:

  • You use Tags & Elements.
  • Tags are brackets "<>" that contain an Element.
  • The Element is the instruction, telling the browser how to display the content.
  • Most all tags have an opening tag & closing tag.
  • Void tags are the exception like meta & img because it has no closing tag.
